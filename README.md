# m01nm01n.github.io

## Requirements
- [pnpm](https://pnpm.io/installation)
- [lefthook](https://github.com/evilmartians/lefthook)
```bash
pnpm install

# setup git hooks
pnpm lefthook install
```
## Lint & Format
```bash
pnpm check
```


## Dev
```bash
pnpm dev
```
## Build
```bash
pnpm build
```
